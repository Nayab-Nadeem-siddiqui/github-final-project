
# This is the README.md file for the **github-final-project**

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

**Input:**
- p, principal amount  
- t, time period in years  
- r, annual rate of interest  

**Output**
- simple interest = p * t * r

## Project Context
This sample repository is part of a micro-finance initiative to empower low-income individuals. It demonstrates Git best practices (license, code of conduct, contribution guide) while hosting a simple script for calculating simple interest.
